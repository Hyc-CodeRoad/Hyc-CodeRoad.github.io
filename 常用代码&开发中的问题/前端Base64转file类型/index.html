2022-06-26 常用代码&开发中的问题 0 评论

功能背景

现有一个域名管理系统，需要上传申请原件（图片）到数据库，现在是将图片以二进制形式直接存储到数据库中。在执行修改操作时，后端返回给前端的为base64格式的照片，如果没有修改原件，重新上传后端使用MultipartFile就无法接收成功。所以需要将base64格式的照片重新转成file格式。
