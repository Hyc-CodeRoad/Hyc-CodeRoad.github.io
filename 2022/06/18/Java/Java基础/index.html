Java基础

Java概述

第一个Java程序

使用记事本或者其他文本编辑工具即可

  1. 将文件后缀名改成.java
  2. 在控制台中使用javac命令对.java文件进行编译，需要带.java后缀
  3. 使用java命令运行生成的.class文件，运行是不需要带.class后缀，直接文件名即可
  • 编译
  • 运行
  • 在一个类中可以声明多个class，但是只有一个类声明为public的，跟文件名相同的类上
  • main方法是程序的入口，Java程序从main方法开始执行
  • 每一行都以分号结尾
  • Java程序严格区分大小写
  • 一个类中声明了多个class，那么编译后会生成多个.class文件，文件名与class后面的类名相同

Java中有三种注释：

  • 单行注释
  • 多行注释
  • 文档注释

基本语法

关键字和保留字

关键字

被Java语言赋予了特殊含有，用做专门用途的单词。

关键字中的所有字母都为小写。

保留字

现有的版本还没有使用，但以后的版本可能会作为关键字使用。

标识符

就是Java中需要对变量，方法和类等需要命名的地方所使用的字符序列。需要自己起名字的地方都叫标识符。

命名规则

  • 由字母，数字，下划线和美元符号组成
  • 数字不可以开头
  • 不可以使用关键字和保留字，但是可以包含
  • Java中严格区分大小写，长度无限制
  • 标识符不能包含空格

命名规范

规范是行业长期形成的规范，可以不遵守。

  • 包名：全部小写
  • 类名：接口名：首字母大写，多个单词是遵循驼峰命名
  • 变量名：方法名：首字母小写，多个单吃时遵循驼峰命名
  • 常量名：所有字母大写，多个单词时，使用下划线隔开
  • 命名时为了提高可阅读性，尽量有意义，见名知意
  • Java使用unicode字符集，因此也可以使用中文或者其他国家字符来声明，不建议使用。

变量

概念

就是内存中的一块存储区域。该区域可以在同一类型范围内不断变化。

变量式程序中最基本的存储单元。包含变量类型，变量名和存储的值。

定义变量

  • 变量必须初始化（赋值）以后才能使用
  • 变量都定义其作用域内，在作用域内都是有效的
  • 同一个作用域内，不可以声明两个同名的变量

数据类型（主要讲基本数据类型）

分类

按数据类型分可以分为：基本数据类型和引用数据类型。

根据在类中的声明位置可以分为：成员变量和局部变量（目前了解即可）

整型

Java中的整数有固定的表述范围，不受操作系统影响，以保证Java程序的可移植性。

类型 空间 大小
byte 1字节 -2的7次~2的7次-1
short 2字节 -2的15次~2的15次-1
int 4字节 -2的31次~2的31次-1
long 8字节 -2的63次~2的63次-1

Java整型常量默认为int类型，声明long类型时常量后需要加L或l。

浮点型

浮点型通常有两种表示形式：

  • 十进制数表示形式
  • 科学计数法表示形式
类型 空间
float（单精度） 4字节
double（双精度） 8字节

如果需要追求精度吗如科学计算，银行存款等等，不能使用float和double来表示，Java提供了专门的类来处理需要高精度的情况。

字符型（char）

char类型就是用来表示通常意义上的字符（2字节）。

Java中的所有字符都使用Unicode编码，所以字符，汉字，或其他语言的一个字符都可以。

字符常量使用单引号引起来的单个字符。

Java中允许使用转义字符，可以直接使用Unicode的值来表示字符型常量。

char类型可以参与运行，因为它都有对应的Unicode码。

布尔类型（boolean）

boolean类型只有true和false两个值。

在Java中不能使用0和1来表示ture和false。

boolean类型不能参与数值计算。

数据类型转换

自动类型转换

自动类型转换：容量小的类型自动转换为容量大的数据类型。

  • 在多种类型参与运算时，系统会自动将所有类型转换为其中容量最大的数据类型，再进行运算。

强制类型转换

强制类型转换：容量大的类型转换为容量小的数据类型时，需要进行强制类型转换

  • 强制类型转换可能会造成精度损失问题
