网站概述

网站概述

网站是指在因特网上根据一定的规则，使用 HTML 等制作的用于展示特定内容相关的网页集合。

网页是网站中的一“页”，通常是 HTML 格式的文件，它要通过浏览器来阅读。

网页是构成网站的基本元素，它通常由图片、链接、文字、声音、视频等元素组成。通常我们看到的网页，常见以 .htm 或 .html 后缀结尾的文件，因此将其俗称为 HTML 文件。

常用浏览器

  • IE
  • 火狐
  • 谷歌
  • Safari（苹果浏览器）
  • Opera（欧朋）

浏览器内核

浏览器内核主要负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。

浏览器 内核 备注
IE Trident 国内大多数浏览器都使用这个
火狐 Gecko
Safari Webkit
谷歌/欧朋 Blink Blink其实是Webkit的分支

Web标准

Web标准是由W3C和其他标准化组织制定的一系列标准的集合。W3C（万维网联盟）是国际最著名的标准化组织。

统一的Web标准不仅可以让开发人员的代码更加规范，还可以：

  1. 让Web发展前景更加广阔
  2. 内容能被更多的设备访问
  3. 更容易被搜索引擎搜索
  4. 降低网站流量费用
  5. 使网站更加容易维护
  6. 提高页面的速度

Web标准主要由：结构，表现，行为三个方面构成：

  • 结构：用于对网页元素进行整理和分类，HTML
  • 表现：用于设置网页元素的样式，CSS
  • 行为：对网页模型的定义以及交互的编写，JavaScript

Web标准提出最佳体验方案：结构，样式，行为相互分离，即：HTML写在html文件中，css写在css文件中，JavaScript写在js文件中。

HTML概述

HTML（Hyper Text Markup Language），指超文本标记语言，是一种用来表述网页的语言。

HTML不是编程语言，而是一种标记语言，即标记标签。

HTML文件通常以.html或.htm结尾。

标签之间的关系可以分为包含和并列关系。如：

  • html标签包含h1和h2
  • h1和h2为并列关系

HTML标签

基本结构标签

每个页面都会有一些基本的结构标签（骨架标签）。

标签名 定义 说明
html HTML标签 页面中最大的标签，也被称为根标签
head 文档的头部 设置网页头部信息
title 文档的标题 设置网页标题，是head中必须要设置的标签
body 文档的主体 文档的主题，页面的内容基本都写在这里面

HTML的基本结构

  • <!DOCTYPE html> 文档类型声明标签，告诉浏览器这个页面采取html5版本来显示页面。
  • lang属性用来定义当前文档的语言，不管定义成什么语言，都是可以显示其他语言的。
  • <meta charset="UTF-8"> 规定HTML文档使用哪种字符集。

标题标签 <h1>~<h6>

  • 一个标题独占一行
  • 文字会加粗，h1刀h6字号逐渐减小

段落标签 <p>

  • 文本在段落标签中会根据浏览器的大小自动换行
  • 段落和段落之间会有空隙

换行标签 <br />

  • 换行标签是一个单标签
  • 就是简单的开始下一行

文本格式化标签

含义 标签
加粗 <strong></strong> 或者<b></b>
倾斜 <em></em>或者<i></i>
删除线 <del><del> 或者<s></s>
下划线 <ins></ins> 或者<u></u>

盒子标签<div>和<span>

div和span是没有语义的就是一个盒子，用来装内容。

  • div标签主要用来布局，一行只能放一个
  • span标签一行可以放多个

图像标签<img>

属性

属性 含义
src 图片路径
alt 替换文本，图片不能正常显示时文字
title 提示文本，鼠标移动到图片上时，显示的文字
width 图像宽度
height 图像高度
border 设置图像的边框
  • 可以同时写多个属性，属性没有先后顺序
  • 属性采用键值对的方式

链接标签<a>

属性

属性 含义 取值
href 跳转的目标 链接，html页面，#（空链接）…
target 用于指定链接页面的打开方式 _self（默认值），_blank新窗口打开

分类：

  1. 外部链接：例如：百度网址
  2. 内部链接，跳转到其他html页面
  3. 空链接：hred值为#
  4. 下载链接：当href的值为一个文件或者压缩包时，会下载这个文件
  5. 网页元素链接：给网页中的各种文本，图像，视频等都可以添加超链接
  6. 锚点链接：点击链接快速定位到页面中的某个位置，将href的值设置为#名字的形式，将目标位置的标签里添加id属性，名字就是这个id属性

注释

注释不会参与代码的执行，为了对代码进行解释说明，提高代码可读性。
